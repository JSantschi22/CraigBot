About CraigBot

CraigBot was built by Jeffrey Santschi for his ultimate team Craig's Cutters and his fellow ultimate coaches for the LCS high school ultimate team. CraigBot is able to answer questions about the official USA ultimate ruleset, as well as the MODS' amendments for play in Manitoba.

You can find the source code by clicking the GitHub link in the header, or by clicking here.

To see the official USA ultimate ruleset you can go to usaultimate.org and to see the MODS' amendments visit mods.mb.ca.

CraigBot was built using Llama Index AI and Groq AI.

This project is entirely open source and just for fun, hope you enjoy. It is possible that an adblocking extension could interfere with the operations of the site. If you are experiencing issues try disabling it and reloading.
